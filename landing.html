Features

Everything you need to run efficient, fair, and measurable queues.

Configurable Services

Create and group services, assign counters, and set priorities.

Kiosk Token System

Self-serve kiosks issue tokens and QR codes for streamlined check-in.

Fine-grained roles for admins, staff, and managers with approval workflows.
